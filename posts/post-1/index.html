Pengertian Sampling

Sampling atau teknik pencuplikan adalah bagian dari metodologi statistika yang berhubungan dengan prosedur sistematis dalam pengambilan sebagian anggota populasi untuk keperluan pendugaan (estimasi). Jika pengambilan sampel dilakukan dengan teknik yang tepat, analisis statistik dari suatu sampel bisa digunakan untuk menggeneralisasikan keseluruhan populasi.

Rumus Sampling dari Ahli Luar Negeri

Rumus Slovin

Rumus Slovin adalah sistem matematis yang digunakan untuk menghitung jumlah sampel dari sebuah populasi objek tertentu yang belum diketahui karakteristiknya secara pasti. Rumus ini diperkenalkan oleh seorang ilmuwan matematis bernama Slovin pada tahun 1960.

Keterangan:

  • ( n ) = ukuran sampel
  • ( N ) = ukuran populasi
  • ( e ) = persen kelonggaran ketidaktelitian karena kesalahan pengambilan sampel yang bisa ditolerir

Rumus Lemeshow

Rumus ini digunakan untuk menentukan besar sampel penelitian, terutama dalam bidang kesehatan masyarakat.

Keterangan:

  • ( n ) = jumlah sampel minimal yang diperlukan
  • ( Z ) = derajat kepercayaan
  • ( P ) = proporsi kasus terhadap populasi
  • ( d ) = derajat penyimpangan terhadap populasi
